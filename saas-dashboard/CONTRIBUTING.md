\# Contributing Guide



Thanks for contributing to Orbit SaaS Dashboard.



\## Development Workflow



1\. Fork repository

2\. Create feature branch

3\. Commit changes

4\. Run tests

5\. Open pull request



\## Coding Standards



\- TypeScript strict mode

\- ESLint compliance

\- Prettier formatting

\- Unit tests for utilities and APIs



\## Branch Naming



Examples:



feature/add-billing-ui

fix/auth-session-bug

refactor/dashboard-layout
